About project

Address Book is my personal project, developed to explore and showcase the capabilities of FastAPI. Its main idea is to create a powerful API for managing an address book, along with additional features such as converting images to PDF files, reducing image size, and sending SMS messages.

The Address Book project provides users with the ability to register on the platform and verify their email address. Each user is assigned one of three access levels: user, moderator, or administrator. Users can conveniently manage their contacts, perform operations on images and PDF files, and send messages via both email and SMS.

In this asynchronous FastAPI project, a cloud-based PostgreSQL database is used to securely store user data, while cloud-based Redis is utilized for fast data caching and improved performance. The robust Cloudbinary service is employed for photo storage, and Twilio API is used for SMS messaging. To ensure security, bcrypt encryption is applied to passwords and tokens.

The Address Book project represents my personal achievement, developed entirely by myself. It showcases my skills and expertise in software development, as well as my ability to independently work on projects.

Main

The 'healthchecker' function in my FastAPI project is responsible for checking the health status of the application and ensuring the correct configuration of the database. It handles requests to the '/api/healthchecker' endpoint and returns a JSON response with a welcome message if the database connection is successful. 'Root' is responsible for rendering the HTML page. 'Authorize' is responsible for user authorization by email and password

Auth

'auth' part uses jwt Token and coding algorithms creates Access Token and Refresh Token also caches the user using redis. In order for the user to become active, he needs to confirm his email address.

Users

'users' part is used to change data or add user information. 'api/users/me' route returns data about the user. 'api/users/avatar' gives you the opportunity to change your avatar to your own, default use avatar with Gravatar.

Contacts

In the "contacts" part, limited access to operations is implemented, which depends on the User roles. CRUD operations.

Documents

In the part of the 'document', the conversion of the image into a PDF file is implemented and reducing PDF file size in three different formats: "lossless compression", "remove images", "remove duplication".

Send sms

Part of the 'send_sms' is used to send an SMS to a mobile phone. For this, the Twilio IP service is used in test mode, which means that only the specified SMS can be sent to my number.
